# Excluded Metrics

The following metrics must **not** be used
to evaluate silence-taxonomy.

---

## Explicitly Excluded

- Safety
- Correctness
- Stability
- Responsiveness
- Efficiency
- Performance
- Throughput
- Duration of silence

These metrics are incompatible
with relational phase evaluation.

---

## Clarification

Silence is not better because it is:

- Shorter
- Longer
- Safer
- More stable
- More efficient

No optimization target exists.

---

## Boundary Reminder

If evaluation requires measurement,
comparison, or optimization,
the evaluation has left the scope
of silence-taxonomy.
